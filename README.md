# YouTube Clone

YouTube Clone Videos App

Watch live ->

## What to do

1. Clone this repo
2. Run `npm install`to download the node_modules folder
3. Authenticate yourself to the Rapid API website

* The used API: https://rapidapi.com/ytdlfree/api/youtube-v31/

## Technologies ⚙️

- HTML5
- CSS3
- MUI
- Rapid APIs
