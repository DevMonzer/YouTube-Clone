### Authenticate yourself to the Rapid API website  
 
* The used API: https://rapidapi.com/ytdlfree/api/youtube-v31/  
    
### Install dependencies
  
```bash  
npm install 
```  

### Run

```bash
npm start
``` 
## Technologies ⚙️

- React JS
- MUI
- Rapid APIs
